\section*{Code availability}
% ------------------------------------------------------------

All source code, the LaTeX sources for this report, and the Jupyter notebook
\texttt{mfpt\_code.ipynb} are publicly available at
\url{https://github.com/NirabHossain/Fokker-Planck-mean-first-passage-time-approach}.
The three numerical methods are stand-alone functions that share a common
spatial discretisation; reproducing any figure requires re-executing only the
corresponding cell.
